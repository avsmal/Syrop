Пример запуска интерпритатора Python из программы на C++ с использолванием
boost_python:

Установка boost в Ubuntu-like системах проще всего осуществляется с помощью:
>sudo apt-get install boost-all-dev
она установит все части boost в том числе и boost_python с зависимостями.
Естественно для системы сборки зависимости придется отследить аккуратнее.

Для использования boost_python в исходном коде подключается заголовок
boost/python.hpp и используется простарнство имен boost::python. Перед
началом использования Python интерпритатора необходимо вызвать функцию:
Py_Initialize()
а по завершению функцию:
Py_Finalize()

Чтобы компилятор нашел необходимые заголовочные файлы Python необходимо
указать компилятору путь к заголовкам Python /usr/include/pythonX.Y , а
линковщику указать использование библиотек libpythonX.Y и libbost_python

Пример смотрите в каталоге src, документация доступна на сайте boost,
например, по адресу
www.boost.org/doc/libs/1_49_0/libs/python/doc/index.html
